Mapeo 3D

Este ejercicio consiste en un mapeo 3D.
En el siguiente lienzo se incluye el modelo (mesh) de un bloque de la Ciudad de México que contiene el munmento d el Ángel de la Independencia.
El modelo fue extraído de Google Maps e integrado a la web utilizando Three.js, una librería de JavaScript para la creación de elementos 3D.
Además, se programaron figuras, texturas e interactividad, permitiendo que al hacer clic en el rectángulo rojo que engloba el monumento, se muestre una capa *div* con un video.
